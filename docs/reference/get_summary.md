# Get Summary

Function for creating a short summary of the estimated Iota components.

## Usage

``` r
get_summary(object)
```

## Arguments

- object:

  An object of class `iotarelr_iota2` created by
  [`compute_iota2`](compute_iota2.md),
  [`check_new_rater`](check_new_rater.md), or
  [`check_dgf`](check_dgf.md).

## Value

Prints central statistics of the estimated model.
