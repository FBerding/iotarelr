# Check assumptions of weak superiority

This function tests if the probabilities within the Assignment Error
Matrix are in line with the assumption of weak superiority.

## Usage

``` r
check_conformity_c(aem)
```

## Arguments

- aem:

  matrix of probabilities

## Value

Returns the number of violations of the assumption of weak superiority.
0 if the assumptions are fulfilled.
