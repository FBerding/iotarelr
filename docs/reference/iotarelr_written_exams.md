# Example Data Set

A data set containing the ratings of three coders for written exams. It
also contains the gender of the people who took the exam. The data is
not real and is only created for illustrating purposes.

## Usage

``` r
iotarelr_written_exams
```

## Format

A data frame with 318 rows and 4 variables:

- Coder A:

  Ratings of coder A.

- Coder B:

  Ratings of coder B.

- Coder C:

  Ratings of coder C.

- Sex:

  Referring to the biological aspects of an individual.
